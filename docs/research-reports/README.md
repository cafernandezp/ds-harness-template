# docs/research-reports/

Prior technical explorations by ADVISOR. Comparisons of methods,
strategies, or models where the tradeoffs matter and the analysis is
worth preserving beyond a single chat.

## Rules

- One file per topic: `<slug>.md`.
- Written exclusively by ADVISOR via the `ds-research-report` skill
  (`skills/ds-research-report/SKILL.md`).
- Always written in English, regardless of chat language.
- Reports document **options explored, no decision**. If a decision
  follows, LEAD writes a separate ADR in `docs/adr/` that cites the
  report — never mix the two in one file.
- Consult before writing on the same topic — extend the existing report
  instead of duplicating it.
